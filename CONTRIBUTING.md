# Contributing

Thanks for looking! This is a small personal tool that's free to use and modify.

## Issues
Bug reports, questions, feature ideas — all welcome. Include:
- What you tried
- What happened vs. what you expected
- OS / GPU / relevant versions

## PRs
- Keep changes focused (one thing at a time)
- Match existing style (Rust: rustfmt; Python: PEP-8 mostly)
- Update CHANGELOG for user-visible changes
- No tests required, but appreciated

## Scope

This repo is maintained as a side tool. Big architectural changes or commercial features are **out of scope**. Fork freely if you need that direction.
